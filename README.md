🌐 AI Language Translator

A modern desktop translation application developed using Python, Tkinter, and Deep Translator. The application allows users to translate text between multiple languages through a clean and intuitive graphical user interface.

---

📌 Project Overview

The AI Language Translator enables users to enter text, select source and target languages, and instantly receive translated output. The application is designed with simplicity, usability, and functionality in mind.

This project was developed as part of the CodeAlpha AI Internship Program.

---

✨ Features

- 🌍 Multi-language translation support
- 🖥️ Interactive Graphical User Interface (GUI)
- 🔄 Language Swap Functionality
- ⚡ Fast and accurate translations
- 🎨 Modern blue-themed interface
- 🛡️ Input validation and error handling
- 📱 Mobile-friendly development using Pydroid 3

---

🛠️ Technologies Used

- Python
- Tkinter
- Deep Translator
- Google Translator Service

---

🚀 Installation

Install Required Package

pip install deep-translator

Run the Application

python language_translator.py

---

📖 Usage

1. Enter text in the input field.
2. Select the source language.
3. Select the target language.
4. Click the Translate button.
5. View the translated output.
6. Use the ⇄ Swap button to switch languages.
